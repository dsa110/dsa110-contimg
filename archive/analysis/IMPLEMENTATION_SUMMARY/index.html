    Known Issues¶

    1. Consolidated pages embed full pages: The consolidated pages currently render the full original pages as children, which may cause duplicate headers/containers. This should be refactored to extract just the content.

    2. Tab navigation in consolidated pages: The embedded pages have their own tabs, which may conflict with the parent page tabs. Need to refactor to extract content only.

    3. TypeScript errors: May need to check for any type errors in new components.
