    Technical Details¶

    Algorithm: Differential Flux Ratios¶

    Baseline establishment: 

    # Use first 10 epochs
for each reference:
    flux_baseline = median(flux[0:10])

    Per-epoch correction: 

    # Measure all references
ratios = [R_i(t) / R_i_baseline for all i]
correction = median(ratios)

# Apply to target
flux_norm = flux_target / correction

    Error propagation: 

    σ_norm² = (σ_meas / corr)² + (F_raw × σ_corr / corr²)²

    Reference Source Criteria¶

    From master_sources.final_references: - NVSS SNR > 50 (or 80 for highest quality) - Spectral index: -1.2 < α < 0.2 (flat spectrum) - Not resolved (compact sources) - No confusion (single source in beam) - Spatially distributed across FoV

    Expected Performance¶

    Without normalization: - Day-to-day scatter: ~5-10% (atmospheric + instrumental)

    With normalization: - Day-to-day scatter: <3% (target: 1-2%) - Enables detection of 10-50% ESE flux variations at 5-10σ
