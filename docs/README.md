# DSA-110 Continuum Imaging Pipeline Documentation

This directory contains all documentation for the DSA-110 continuum imaging pipeline. The documentation is organized into the following categories:

- **[Concepts](concepts/index.md)**: High-level explanations of the pipeline's architecture, design, and core concepts.
- **[Tutorials](tutorials/)**: Step-by-step guides to performing common tasks.
- **[How-To Guides](how-to/)**: Detailed instructions for specific procedures.
- **[Reference](reference/)**: Technical reference material, such as API endpoints, database schemas, and CLI commands.
- **[Operations](operations/)**: Deployment, operations, and troubleshooting guides.
- **[Development Notes](dev/)**: Development notes, status reports, analysis, and history (for developers and AI agents).
- **[Contributing](contributing/)**: Guidelines for contributing to the project.

## Quick Start

- **New to the pipeline?** Start with [Concepts](concepts/pipeline_overview.md) and [Quickstart Guide](how-to/quickstart.md)
- **Looking for a specific task?** Check [How-To Guides](how-to/)
- **Need API or CLI reference?** See [Reference Documentation](reference/)
- **Developing or debugging?** See [Development Notes](dev/)

## Documentation Organization

### User-Facing Documentation
- **Concepts**: Understand how the pipeline works
- **How-To**: Complete specific tasks
- **Tutorials**: Learn through step-by-step examples
- **Reference**: Look up API endpoints, CLI commands, schemas
- **Operations**: Deploy and operate the pipeline

### Development Documentation
- **Status Reports**: Current status of features and components
- **Analysis**: Investigation reports and technical deep-dives
- **Notes**: Agent notes and brainstorming
- **History**: Completed investigations and development history

See [Documentation Consolidation Strategy](DOCUMENTATION_CONSOLIDATION_STRATEGY.md) for details on the documentation structure.
